Command What it does 
chsh -bin /bin/bash
 Change to bash shell (mainly needed on mac computers) 
pwd
 Print directory pathway to show where you are 
ls
 List files and directories in your current directory 
ls -F
 List files with file type indicated 
cd <directory name>
 Change to a specific directory 
mkdir <name of file>
 Make a new directory 
cd ..
 Go up one directory 
cd ~
 Go to home directory 
cd \
 Go to root directory 
cd -
 Go to previous directory 
rm <file name>
 Permanently delete a file or directory 
cp <original file name> <new file name>
 Create a copy of a file 
ssh <IP>
 Connect to remote server, IP: root@##.##.###.### 
scp <location><file path> .

(space period at the end is very important)

Transfer a file from server to your local computer

eg. scp root@10.10.100.100:/data/folder/file.text .

 
scp file.txt remote_username@10.10.0.2:/remote/directory .

Transfer a file from your local computer to the server

eg. scp file.txt root@10.10.100.100:/data/folder .

 
touch
 Create a file

Note: < > are not included in the command; they indicate areas to input specific information
